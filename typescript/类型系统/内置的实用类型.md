### 核心实用类型

#### Awaited\<Type>
可以解包Promise的结果类型
```typescript
type A = Awaited<Promise<string>>; // type A = string
type B = Awaited<Promise<Promise<number>>>; // type B = number
type C = Awaited<boolean | Promise<number>>; // type C = number | boolean
```

#### Partial\<Type>  
将类型 Type 的所有属性设置为可选
```typescript
interface User { name: string; age: number; }
type PartialUser = Partial<User>; // 等价于 { name?: string; age?: number; }
```

#### Required\<Type>
将类型 Type 的所有属性设置为必需
```typescript
interface Props {   name?: string;   age?: number; }
type RequiredProps = Required<Props>; // 等价于 { name: string; age: number; }
```

#### Readonly\<Type>
将类型 Type 的所有属性设置为只读
```typescript
interface Todo {   title: string; }
type ReadonlyTodo = Readonly<Todo>; // 等价于 { readonly title: string; }
```

#### Record\<Keys, Type>
构造一个对象类型，其属性键为 Keys，属性值为 Type
```typescript
type PageInfo = {   title: string; };
type Page = "home" | "about" | "contact";

const nav: Record<Page, PageInfo> = {
	home: { title: "Home" },
	about: { title: "About" },
	contact: { title: "Contact" }, 
};
```

#### Pick<Type, Keys>
从类型 Type 中选取一组属性 Keys 来构造类型
```typescript
interface Todo {
	title: string;
	description: string;
	completed: boolean;
}
type TodoPreview = Pick<Todo, "title" | "completed">; // 等价于 { title: string; completed: boolean; }
```

#### Omit<Type, Keys>
从类型 Type 中排除一组属性 Keys 来构造类型
```typescript
interface Todo {
	title: string;
	description: string;
	completed: boolean;
}
type TodoPreview = Omit<Todo, "description">; // 等价于 { title: string; completed: boolean; }
```

#### Exclude<UnionType, ExcludedMembers>
从联合类型 UnionType 中排除可分配给 ExcludedMembers 的类型
```typescript
type T0 = Exclude<"a" | "b" | "c", "a">; // 等价于 "b" | "c"
type T1 = Exclude<string | number | (() => void), Function>; // 等价于 string | number
```

#### Extract<Type, Union>
从类型 Type 中提取可分配给 Union 的类型
```typescript
type T0 = Extract<"a" | "b" | "c", "a" | "f">; // 等价于 "a"
type T1 = Extract<string | number | (() => void), Function>; // 等价于 () => void
```

#### NonNullable\<Type>
从类型 Type 中排除 null 和 undefined
```typescript
type T0 = NonNullable<string | number | undefined>; // 等价于 string | number
type T1 = NonNullable<string[] | null | undefined>; // 等价于 string[]
```

#### Parameters\<Type>
从函数类型 Type 的参数中使用的类型构造一个元组类型
```typescript
type T0 = Parameters<() => string>; // 等价于 []
type T1 = Parameters<(s: string) => void>; // 等价于 [string]
type T2 = Parameters<<T>(arg: T) => T>; // 等价于 [unknown]
```

#### ConstructorParameters\<Type>
从构造函数类型的类型构造元组或数组类型
```typescript
type T0 = ConstructorParameters<ErrorConstructor>; // 等价于 [message?: string]
type T1 = ConstructorParameters<FunctionConstructor>; // 等价于 string[]
```

#### ReturnType\<Type>
由函数类型 Type 的返回类型构造一个类型
```typescript
type T0 = ReturnType<() => string>; // 等价于 string
type T1 = ReturnType<(s: string) => void>; // 等价于 void
```

#### InstanceType\<Type>
由构造函数类型 Type 的实例类型构造一个类型
```typescript
type T1 = InstanceType<any>; // 等价于 any

class C {
	x = 0;
	y = 0;
}
type T0 = InstanceType<typeof C>; // 等价于 C
```

#### NoInfer\<Type>
‌**版本：5.4**
阻止类型推断，可以确保类型检查更加严格和可预测
```typescript
function createStreetLight<C extends string>(
	colors: C[],
	defaultColor?: NoInfer<C>,
) {
	// ...
}
createStreetLight(["red", "yellow", "green"], "red"); // ✅ 正确
createStreetLight(["red", "yellow", "green"], "blue"); // ❌ 错误，“blue”不在泛型C中
```

#### ThisParameterType\<Type>
提取函数类型的 this 参数类型
```typescript
function toHex(this: Number) {
	return this.toString(16);
}
type T0 = ThisParameterType<typeof toHex>; // 等价于 Number
```

#### OmitThisParameter\<Type>
从类型 Type 中移除 this 参数
```typescript
function toHex(this: Number) {
	return this.toString(16);
}
type T0 = OmitThisParameter<typeof toHex>; // 等价于 () => string
```

#### ThisType\<Type>
在对象字面量中指定 this 的上下文类型
```typescript
type ObjectDescriptor<D, M> = {
	data?: D;
	methods?: M & ThisType<D & M>; // M对象中的this指向data数据+methods数据
};

const obj: ObjectDescriptor = {
	data: { x: 0, y: 0 },
	methods: {
		moveBy(dx: number, dy: number) {
		this.x += dx; // this.x类型等于obj.data.x
		this.y += dy; // this.y类型等于obj.data.y
	}
}
```

### 字符串操作类型‌

#### Uppercase\<StringType>
将字符串中的每个字符转换为大写
```typescript
type Greeting = "Hello, world";
type ShoutyGreeting = Uppercase<Greeting>; // 等价于 "HELLO, WORLD"
```

#### Lowercase\<StringType>
将字符串中的每个字符转换为小写
```typescript
type Greeting = "HELLO, WORLD";
type QuietGreeting = Lowercase<Greeting>; // 等价于 "hello, world"
```

#### Capitalize\<StringType>
将字符串中的第一个字符转换为大写
```typescript
type LowercaseGreeting = "hello, world";
type Greeting = Capitalize<LowercaseGreeting>; // 等价于 "Hello, world"
```

#### Uncapitalize\<StringType>
将字符串中的第一个字符转换为小写
```typescript
type UppercaseGreeting = "HELLO, WORLD";
type UncomfortableGreeting = Uncapitalize<UppercaseGreeting>; // 等价于 "hELLO, WORLD"
```
