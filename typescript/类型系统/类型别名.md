1. 为类型声明新的名称
2. 关键字：type

### 基本使用
```typescript
	type User = { name: string, age: number };
	type MyString = string;
```

### 联合类型
1. 由两个或多个类型组合而成的新类型，其值是其中任意的一种类型
2. 如果所有类型都有同一个属性，则使用该属性不需要确认具体属于哪个类型

```typescript
// 联合类型使用 | 分割每个类型
type List = OrderList | DefaultList; 
```

### 交叉类型
1. 获取两个或多个类型的并集

```typescript
// 联合类型使用 & 分割每个类型
type List = AList & BList;
```
