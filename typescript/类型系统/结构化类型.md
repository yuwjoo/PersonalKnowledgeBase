1. 也叫对象类型
### 定义方式
```typescript
// 1. 使用类型别名
type ObjType = {
	name: string,
	age: number
}

// 2. 使用接口
interface ObjType {
	name: string,
	age: number
}
```

### 属性修饰符
```typescript
interface SomeType {
	readonly prop: string; // 属性只读
	age?: string; // 属性可选
}
```

### 索引签名
1. 描述可能属性值的类型，而不需要关心具体有哪些属性
```typescript
interface StringArray {
	[index: number]: string;
}
```
